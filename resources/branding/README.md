# Civora Nexus Branding Assets (EduVillage)

This folder contains official Civora Nexus brand assets used by the EduVillage project.

Structure:
- `logos/` — primary and secondary logos (SVG/PNG)
- `favicons/` — favicon files and metadata
- `social-icons/` — icons for social sharing
- `colors/` — color palette and tokens

Guidelines:
- Follow Civora Nexus branding guidelines strictly.
- Do not add third-party UI libraries or themes in Phase-1.
