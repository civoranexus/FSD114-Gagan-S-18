# EduVillage Architecture

## Phase-1: Project Structure Setup

This document outlines the architecture and folder organization for the EduVillage Online Learning Platform.

### Backend Structure
- **apps/users/** — User management app (placeholder)
- **apps/courses/** — Course management app (placeholder)
- **apps/assessments/** — Assessment/quizzes app (placeholder)
- **apps/notifications/** — Notification system app (placeholder)

### Frontend Structure
- **src/components/** — Reusable UI components
  - common/ — Shared components
  - layout/ — Layout components
- **src/pages/** — Page-level components
  - auth/ — Authentication pages
  - student/ — Student dashboard pages
  - teacher/ — Teacher dashboard pages
  - admin/ — Admin pages
- **src/services/** — API service layer
- **src/styles/** — Global styles
